# 🚀 Current Project

## Developer Project Manager

### Why did I build this?

I'm currently working on multiple personal and professional projects.

While GitHub is excellent for source control, I realized it wasn't enough to organize everything related to my projects. I needed a place where I could centralize all the technical and organizational information that developers work with every day.

After trying several project management tools, I found that they either included too many unnecessary features or lacked the technical details I actually needed.

So I decided to build my own.

# ✨ Features

## 📁 Project Management

Each project stores all relevant information in one place:

* Name
* Description
* Status
* Priority
* Technologies
* Git repositories
* External services (Clerk, ImageKit, Stripe, etc.)
* Deployment platforms (Vercel, Railway, Netlify...)
* Folder structure documentation
* Favorite projects
* Custom icon
* Custom color
* Start date
* End date

## 🧩 Module Management

Projects are divided into modules to organize development.

Each module contains:

* Name
* Description
* Status
* Priority
* Icon
* Color

Example modules:

* Authentication
* Products
* Orders
* Dashboard
* Reports
* Billing
* Notifications

## ✅ Task Management *(In Progress)*

Every module will contain tasks that help organize development.

Each task will include:

* Title
* Description
* Status
* Priority
* Due date
* Estimated time
* Related feature or bug
* Progress tracking

## 📌 Subtasks *(Planned)*

Tasks will be divided into smaller subtasks, making it easier to track development progress and complete complex features step by step.

# 💡 Philosophy

I don't want to build another Jira or Trello.

My goal is to create a lightweight project manager designed specifically for developers.

Instead of focusing on generic business workflows, this application centralizes everything a developer needs while building software:

* Project information
* Modules
* Tasks
* Bugs
* Git repositories
* External services
* Deployment platforms
* Architecture documentation

Everything related to software development in one place.
